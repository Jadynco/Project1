AIStudyTool

Description

This is an Ai powered website that generates flashcards and quizzes from DOCX files, PDF files, or direct text input. The site uses the Open AI API to summerize content and convert to study format to help students learn efficiently. Deployed via vercel and render.

Skills Learned

  • How to intergrate Node.js with React.js for backend functions
  • Intergate APIs (openAI in particular) into websites using Node.js and React.js
  • Deploymenting website frontend and backend using vercel and render
