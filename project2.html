Pomodoro Productivity App

Description

This is a project where I built a pomodoro based productivity tool in React that helps users manage work rest intervals along with a regular timer and stopwatch. Features include adjustable timers, start/pause, and session tracking.

Skills Learned

  • Keeping track of time using react state components
  • Using design features that promote the goal of your website
  • Using react to create an all in one page format
